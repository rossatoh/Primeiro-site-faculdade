PERGUNTAS MAIS FREQUENTES

 Quero fazer um software para minha imobiliária. O que devo fazer?
R: Contate um de nossa equipe para ser instruido, para ter todas
as informações necessárias com mais profundidade em que você
precisa para sua imobiliária.

 Quanto tempo demora para desenvolver o software?
R: Logo após termos todos os detalhes de como o software é desejado,
passaremos as informações para nossa área de desenvolvimento para
entrar em ação, podendo levar de 4 a 9 dias.

 Se, por acaso, o software parar de funcionar, o que a empresa possuidora
precisa fazer?
R: Caso isso ocorra, rapidamente entre em contato com a RE Project,
pois temos um controle de funcionamento dos softwares que criamos
e todos possuim um software de localização e de desempenho para
facilitar na hora de atualizações e casos de falhas.

 O sistema é seguro diante a uma invasão?
R: Nossos profissionais trabalham arduamente para que os arquivos e
documentos que possuem gravados no software possa ser protegido
caso tenham o site hackeado.

Outra pergunta? Mande para nós!
